Jedes Level = fester Seed + Budgets + Punkteziel; das Ziel schlagen schaltet das nächste frei. Alte Level antippen wählt sie an (indigo Rahmen, hier Level 5) — ihr „Spielen" ist grün = Wiederholen (den eigenen Rekord schlagen), während das nächste Level indigo spielt. So unterscheidet die Farbe „weiter" von „nochmal". Schwieriger wird's über knappere Budgets und höhere Ziele — nie über Zeitdruck. Levelnamen erzählen nebenbei die kleine Fuchsgeschichte.
