# Knobelfuchs 🦊🔢

A **calm number-match puzzle** — the classic pen-and-paper *Zahlenknobeln* / Take Ten
mechanic, built the fuchs way: **ad-free, local-first, no lives, no timers, no
paywalled hints, no dark patterns.** Tablet-first, made for a retired parent and
everyone else who just wants to knobel in peace.

The first **game** in the fuchs family, inheriting the shared
[Fuchsbau](https://github.com/Kemenor/fuchsbau) ethos, design system, and tech stack.

## The game

Cross out pairs of digits that are **equal or sum to 10**, along rows, columns,
diagonals, or in reading order — cleared cells are see-through. Empty rows collapse.
Deal more digits whenever you like (free, unlimited — imagine that). Clear the whole
board to win. Hints and undo are unlimited, because assists are part of the game, not
inventory to sell back to you.
